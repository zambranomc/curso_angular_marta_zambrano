Instalación
Instalación global, en cualquier terminal:

npm install -g @nestjs/cli

Comandos nestjs/cli
https://docs.nestjs.com/cli/overview

nest --help

Creación proyecto
nest new backend01 --skip-git --package-manager npm

Ejecutar proyecto
Navegar hacia el proyecto:

npm run start:dev

o

nest start --watch
